Functionality of Application

-- Users must be able to register an account
-- Users must be able to login
    - We want to authenticate our users
-- Users should be able to log a meal
-- Users should be able to log a workout
-- User should be able to view all logged meals
-- User should be able to view all logged workouts
-- Users should be able to view their log history stats 
    (How many logs, how many calories burned, how many calories consumed, etc...)
-- When a User registers, they can apply to be on a free trial of app (For 1 week, or more...)



Optional Functionality
